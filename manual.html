Getting Started

Welcome to the Game Creator! To create your game, follow these steps:

1. Adding a Character

To start, drag a character (Cat,Goborender,Nano,Blue Cat,Giga) from the Asset Library and drop it into the Workspace.

5. Changing the Workspace Background

You can easily change the background of the workspace to make your game more immersive!

  • Locate the Forest asset in the Asset Library.
  • Click and drag the Forest asset into the workspace.
  • Drop it into the workspace area, and it will set the background to a beautiful forest scene!

This feature allows you to personalize your workspace and create an engaging environment for your character to interact with.

2. Using Controls

Once you have a character in the workspace, you can control it using the following options:

  • Move Left/Right/Up/Down: Enter a distance and click the corresponding button.
  • Turn Left/Right: Enter an angle and click the button to turn your character.
  • Loop Operations: Select operations (e.g., move, turn) to repeat based on user-defined iterations.
  • Variables: Set, change, and show variables to control game elements.
  • Script Commands: Write commands directly to control your character.

3. Loop Operations

You can create loops to repeat actions, but first, you need to enter values for the following controls:

  • Move Up
  • Move Down
  • Move Left
  • Move Right
  • Turn Left
  • Turn Right

4. Working with Variables

You can create variables to store values that control aspects of your game:

  • Set Variable: Input a variable name and a value.
  • Change Variable: Modify the value of an existing variable.
  • Show Variable: Display the current value of the variable.

5. Loading Workspace

Use the Load Workspace option to retrieve previously saved configurations. The Undo and Redo buttons help manage changes made during your session.

6. Writing Scripts

In the Scripts section, you can manually write commands. Examples include:


                loop 5 move right
                move right 10
                move left 10
                move up 10
                move down 10
                change-size 100
                wait 5
                say hello
                think thinking

This allows for greater flexibility and control over your character's actions.

4. Saved Actions

In the Saved Actions area, you can view all the operations you have performed. This section helps you keep track of your actions and modify them if needed.

  • You can see a list of all operations you've executed, such as moving or turning.
  • If you mistakenly added an action that you don't want, you can delete it using the Delete button next to the action.

Additionally, you can rearrange your actions:

  • Simply drag and drop operations to reorder them. For example, if you want Move Down to come after Move Left, just drag it into place!

This feature allows you to customize your actions easily and make your character perform tasks in the exact order you want.

7. Running Your Game

After setting up your character, controls, loops, and variables, click the Play button to see your character execute the operations you've defined.

Tips

  • Experiment with different movements and actions to see how your character behaves.
  • Use loops for repetitive actions to simplify your script.
  • Save your workspace frequently to avoid losing your progress.
